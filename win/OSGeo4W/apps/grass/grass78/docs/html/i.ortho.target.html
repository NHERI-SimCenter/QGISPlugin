NAME

i.ortho.target - Select or modify the imagery group target.

KEYWORDS

imagery, orthorectify

SYNOPSIS

i.ortho.target
i.ortho.target --help
i.ortho.target group=name target_location=name mapset_location=name [--help] [--verbose] [--quiet] [--ui]

Flags:

--help
Print usage summary
--verbose
Verbose module output
--quiet
Quiet module output
--ui
Force launching GUI dialog

Parameters:

group=name [required]
Name of imagery group for ortho-rectification
target_location=name [required]
Location name
Name of target location for ortho-rectification
mapset_location=name [required]
Name of mapset (default: current search path)
Name of target mapset for ortho-rectification
