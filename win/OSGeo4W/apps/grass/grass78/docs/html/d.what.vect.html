NAME

d.what.vect.py - Allows the user to interactively query vector map layers at user-selected locations.

KEYWORDS

display, vector

SYNOPSIS

d.what.vect.py
d.what.vect.py --help
d.what.vect.py map=name[,name,...] [--help] [--verbose] [--quiet] [--ui]

Flags:

--help
Print usage summary
--verbose
Verbose module output
--quiet
Quiet module output
--ui
Force launching GUI dialog

Parameters:

map=name[,name,...] [required]
Name of input vector map(s)
Or data source(s) for direct OGR access

Table of contents

  • DESCRIPTION
  • SEE ALSO

DESCRIPTION

d.what.vect d.what.vect outputs the category value(s) associated with user-specified location(s) in user-specified vector map layer(s). The active monitor can be selected with d.mon.
